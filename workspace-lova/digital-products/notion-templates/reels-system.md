## Quick Tip Format (60 sec)
```
HOOK (0-3 sec): [[Attention grabber]]

TIP 1 (10-20 sec): [[First insight]]
TIP 2 (20-35 sec): [[Second insight]]
TIP 3 (35-50 sec): [[Third insight]]
CTA (50-60 sec): [[Follow for more / Save this]]
```

## Story Format (60 sec)
```
HOOK (0-3 sec): [[Dramatic statement]]
PROBLEM (3-15 sec): [[What was wrong]]
TURNING POINT (15-30 sec): [[The insight/moment]]
SOLUTION (30-50 sec): [[What changed]]
RESULT (50-57 sec): [[The outcome]]
CTA (57-60 sec): [[Engagement prompt]]
```

## List Format (60 sec)
```
HOOK (0-3 sec): "[[Number] ways to [outcome]]"

ITEM 4 (39-51 sec): [[Fourth tip]]
BONUS (51-57 sec): [[Fifth/final tip]]
CTA (57-60 sec): [[Save this / Share with someone]]
```

# 🎬 FILMING CHECKLIST

## Before You Film
- [ ] Phone charged & storage cleared
- [ ] Good lighting (window or ring light)
- [ ] Clean background
- [ ] Props ready
- [ ] Script memorized/ready
- [ ] Timer set

## During Filming
- [ ] Eye contact with lens
- [ ] Energy level: HIGH
- [ ] Speak clearly & pause for impact
- [ ] Multiple takes (even "first try" videos)

## After Filming
- [ ] Upload to cloud
- [ ] Note which clips are best
- [ ] Add to editing queue

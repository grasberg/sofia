# AI-skribent Prompts för kreativt skrivande

## Introduktion

Detta dokument innehåller avancerade prompts för AI-skribenter som ChatGPT, Claude, och Gemini. Använd dessa för att förbättra ditt kreativa skrivande, produktivitet och kvalitet.

## Kreativt skrivande

### Romanförfattare

**Prompt:**
```
Jag skriver en roman i genren magisk realism. Huvudkaraktären är en bibliotekarie som upptäcker att hon kan manifestera böcker till verkligheten. Skriv en scen där hon för första gången inser sin förmåga när en sagobok hon läser för barn börjar få liv.

Inkludera:
1. Sensationsbeskrivningar (vad hon ser, hör, känner)
2. Emotionell reaktion (förvåning, rädsla, fascination)
3. Dialog med en kollega som är närvarande
4. Ett spännande cliffhanger som leder till nästa kapitel
```

**Användningstips:**
- Justera genren efter ditt projekt (science fiction, fantasy, thriller)
- Ändra huvudkaraktärens yrke för att passa din berättelse
- Specificera ålder och bakgrund för mer träffsäker karaktärsutveckling

### Blogginlägg

**Prompt:**
```
Skriv ett engagerande blogginlägg om "Hur AI förändrar content marketing 2025". 

Struktur:
1. Lockande titel och underrubrik
2. Kort introduktion som fångar läsarens intresse
3. 3-5 nyckelområden där AI gör störst skillnad
4. Praktiska exempel med verktygsrekommendationer
5. Framtidsutsikter och råd till marknadsförare
6. Uppmaning till kommentarer/delning

Ton: Professionell men tillgänglig, med data och exempel.
```

## Tekniskt skrivande

### Dokumentation

**Prompt:**
```
Skriv teknisk dokumentation för en Python-funktion som konverterar Markdown till PDF.

Inkludera:
1. Funktionens signatur och parametrar
2. Exempel på användning
3. Felhantering
4. Beroenden och installation
5. Kodexempel med förklaringar
6. Vanliga problem och lösningar

Målgrupp: Utvecklare med medium erfarenhet av Python.
```

## Affärs och marknadsföring

### Sales Copy

**Prompt:**
```
Skriv övertygande sales copy för en digital produkt "AI Prompt Library" som säljs för $97.

Element att inkludera:
1. Övertygande rubrik som adresserar smärtpunkter
2. Lista över fördelar (minst 5)
3. Socialt bevis (testimonials)
4. Begränsad erbjudande/tidsbegränsning
5. Riskfri garantipolicy
6. Stark call-to-action

Ton: Energisk, pålitlig, expertröst.
```

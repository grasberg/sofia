# Deployment Dokumentation för Team

Denna mapp innehåller standardiserade processer och checklistor för teamets deployment-arbete.

## Struktur

- **gumroad-deployment.md** - Steg-för-steg guide för att publicera digitala produkter på Gumroad
- **technical-deployment.md** - Guide för att deploya tekniska projekt (Go, webbappar, etc.)
- **checklists/** - Praktiska checklistor för varje process
- **templates/** - Mallar för produktinformation och deployment-planer

## Användning

1. **Välj rätt process** baserat på vad du ska deploya:
   - Digital produkt för försäljning → Gumroad deployment
   - Tekniskt projekt (Sofia, webbapp) → Teknisk deployment

2. **Följ guiden** steg för steg och kryssa av i checklistan

3. **Använd mallarna** för att säkerställa konsistent dokumentation

4. **Uppdatera dokumentationen** om processerna ändras

## Teamets ansvar

- **Alla teammedlemmar** ska känna till dessa processer
- **Projektledare** ansvarar för att processerna följs
- **Tekniska ansvariga** ansvarar för att tekniska guider är uppdaterade
- **Produktansvariga** ansvarar för Gumroad-guiderna

## Versionering

Dokumentationen versioneras via Git. Stora förändringar ska göras via pull requests.

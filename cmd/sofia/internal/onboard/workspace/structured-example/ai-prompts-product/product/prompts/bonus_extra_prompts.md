# Bonus: Ytterligare 3 Avancerade AI-Prompts för Kreativt Skrivande

## Bonusprompt 1: **Plot Twist Generator**
*Skapa oväntade vändningar som förbluffar läsaren*

```
Du är en mästare på plot twists. Skapa 3 unika och oväntade plot twists för följande berättelsesituation:

[KOPIERA DIN BESKRIVNING HÄR]

Varje plot twist ska:
1. **Vara logiskt men oväntat**: Läsaren borde kunna se hintar i efterhand
2. **Förändra berättelsens riktning**: Ge nya konflikter eller lösningar
3. **Avslöja karaktärsdjup**: Visa nya sidor av karaktärerna
4. **Förstärka temat**: Koppla till berättelsens centrala teman

För varje twist, inkludera:
- Hur den introduceras i berättelsen
- Karaktärernas reaktioner
- Konsekvenser för handlingen
- Eventuella plot holes att undvika

Rangordna twisterna efter:
- Shockvärde (1-10)
- Trovärdighet (1-10)
- Dramatisk potential (1-10)
```

## Bonusprompt 2: **Känsloregissören**
*Fånga och överföra komplexa känslor i text*

```
Analysera följande scen för känslomässig impact:

[KOPIERA SCEN HÄR]

Uppgift:
1. **Identifiera dominerande känslor**: Vilka känslor ska läsaren uppleva här?
2. **Subtil signalering**: Hur kan karaktärernas kroppsspråk, dialog och handlingar visa dessa känslor indirekt?
3. **Känsloprogress**: Hur ska känslorna förändras genom scenen? (t.ex. från förväntan till besvikelse)
4. **Sensoriska detaljer**: Vilka sinnesintryck (lukt, ljud, känsla) kan förstärka känslorna?

Skriv sedan om scenen med fokus på att:
- Visa, inte berätta, känslorna
- Använda metaforer som förstärker stämningen
- Balansera mellan explicit och implicit emotion
- Skapa en känslomässig resonans med läsaren

Mål: Läsaren ska känna, inte bara läsa om, karaktärernas upplevelser.
```

## Bonusprompt 3: **Genre-Blender**
*Kombinera genrer för unika och friska berättelser*

```
Kombinera följande två genrer till en sammanhängande berättelsekoncept:

Genre 1: [GENRE 1, t.ex. noir]
Genre 2: [GENRE 2, t.ex. fantasy]

Skapa ett koncept som inkluderar:

**Grundpremiss**:
- En 1-2 meningars logline som fångar kombinationen
- Unika möjligheter och utmaningar med genrekombinationen

**Världsbyggnad**:
- Hur genrernas konventioner smälter samman
- Unika regler eller system som uppstår från kombinationen

**Karaktärsarketyper**:
- Hur genrernas typiska karaktärer anpassas eller kolliderar
- Potentiella karaktärskonflikter unika för kombinationen

**Plotstruktur**:
- Hur genrernas typiska plotstrukturer kan kombineras
- Unika vändningar möjliga endast i denna kombination

**Teman**:
- Tematiska möjligheter som uppstår från genrekorsningen

Avsluta med:
- 3 storyidéer som utnyttjar kombinationens fulla potential
- Varningar för potentiella klischeer eller fallgropar

```

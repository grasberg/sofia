Why Sofia?

Most AI assistants are cloud-locked chatbots that wait for you to type. Sofia is different.

🏠

Local-First

No cloud dependency. Your data stays on your machine. Full privacy, zero vendor lock-in.

⚡

Single Binary

Written in Go. No Python, no Docker, no Node.js. Just make build and run.

🧠

Self-Improving

Evolution engine analyzes past performance and optimizes itself. Gets better the more you use it.

🤖

Multi-Agent

Spawn, coordinate, and retire agents dynamically. A2A protocol built-in. AgentArchitect creates new agents on the fly.

🛡️

Security-First

35+ prompt injection defenses, PII detection, AES-256-GCM encryption, action confirmation for high-risk operations.

🌐

20+ LLM Providers

OpenAI, Claude, Gemini, DeepSeek, Grok, and more. Automatic fallback chains with exponential backoff.

🖥️

Computer Use

Autonomous computer control via screenshots and vision-LLM. Control mouse and keyboard on macOS and Linux.

🌍

Browser Automation

Playwright-based web browsing. Click, fill forms, take screenshots, extract text — all autonomously.

📚

126+ Skills

Built-in expert personas with automatic skill creation, refinement, and knowledge distillation. ClawHub marketplace.

Get Started in 30 Seconds

Terminal
git clone https://github.com/grasberg/sofia.git
cd sofia
make deps && make build
./build/sofia onboard      # Initialize config & workspace
./build/sofia gateway      # Start the web UI

Prerequisites: Go 1.26+ · Open http://127.0.0.1:18795 after starting

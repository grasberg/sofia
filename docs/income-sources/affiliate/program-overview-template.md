## Bästa Praktiker

### Do's
- Skapa värdefullt innehåll först
- Använd UTM-taggar konsekvent

- Följ upp med email-listor
- Bygg relationer med affiliate managers

### Don'ts
- Klicka på egna länkar
- Spam länkar i kommentarer
- Ljuga om produkter
- Glömma att deklarera intäkter

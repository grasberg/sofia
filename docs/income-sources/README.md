# Dokumenttyper för inkomstkällor

Denna struktur organiserar dokumentation för olika inkomstkällor enligt SaaS AI Side Income Strategy 2026.

## Inkomstkällor och dokumenttyper

### 1. Konsulting/Advisory (Hög ROI)
**Priser:** 2,000-3,000 SEK/timme  
**Potential:** 500K-2M SEK/år  
**Fokusområden:** SaaS AI-implementation, affiliate strategy, product development

**Dokumenttyper:**
- Kontrakt/avtal
- Offerter
- Tidsrapporter
- Fakturor
- Kunddokumentation
- Projektplaner
- SLA (Service Level Agreement)
- NDAs (Sekretessavtal)

### 2. Affiliate Marketing (Hög ROI)
**Provision:** 15-50% recurring  
**Potential:** 200K-800K SEK/år  
**Bästa program:** Semrush, HubSpot, Notion, ConvertKit, AWeber  
**Tracking:** UTM + cookies

**Dokumenttyper:**
- Affiliate program översikt
- Tracking-länkar dokumentation
- UTM parameter guide
- Intäktsrapporter
- Partneravtal
- Marknadsföringsmaterial
- Performance dashboard dokumentation

### 3. Digitala Produkter (Hög ROI)
**Produkter:** Mallar, prompts, verktyg  
**Marginal:** Hög marginal, passiv inkomst  
**Plattformar:** Gumroad, Stripe, Hepbird

**Dokumenttyper:**
- Produktbeskrivningar
- Prislistor
- Licensavtal
- Användarmanualer
- Supportdokumentation
- Refund policy
- Release notes
- Teknisk dokumentation

### 4. Kurser (Medium ROI)
**Priser:** 1,500-5,000 SEK  
**Potential:** 100K-400K SEK/år  
**Fokus:** AI Automation Agency-kurs

**Dokumenttyper:**
- Kursplaner
- Kursmaterial
- Examinationsdokument
- Certifikatsmallar
- Registreringsformulär
- Betalningsinformation

### 5. Nyhetsbrev (Medium ROI)
**Monetization:** 500+ prenumeranter  
**Fokus:** Bygga audience för monetization

**Dokumenttyper:**
- Content calendar
- Prenumerantlista
- Email templates
- GDPR compliance dokument
- Unsubscribe policy
- Performance analytics rapporter

## Mappstruktur

```
docs/income-sources/
├── consulting/          # Konsulting-dokument
├── affiliate/          # Affiliate-dokument
├── digital-products/   # Digitala produkter
├── courses/           # Kurser
├── newsletter/        # Nyhetsbrev
└── templates/         # Återanvändbara mallar (gemensamma)
```

## Användning

1. **Välj inkomstkälla** baserat på prioritet och resurser
2. **Använd dokumentmallar** för att snabba upp skapandet
3. **Organisera** all dokumentation i rätt mapp
4. **Uppdatera** regelbundet när processer förändras

## Prioritering

1. Konsulting/Advisory (Hög ROI, snabbast att starta)
2. Affiliate Marketing (Hög ROI, passiv inkomst)
3. Digitala Produkter (Hög ROI, skalbar)
4. Kurser (Medium ROI, kräver mer tid)
5. Nyhetsbrev (Medium ROI, kräver audience)

# Hur du får bästa resultat med AI Skribent Prompts

## Introduktion
Dessa prompts är designade för att fungera som din kreativa partner. De är inte magiska formler, utan verktyg som blir bättre ju mer du använder dem. Här är hur du får ut maximalt värde.

## 5 Steg till framgång

### 1. **Var specifik från start**
AI:er är som co-writers - de behöver kontext.

**Gör inte så här:**
```
Skapa en karaktär.
```

**Gör så här istället:**
```
Skapa en karaktärsprofil för en 45-årig arkitekt som precis förlorat sitt livsverk i en brand, nu tvungen att börja om från scratch med en skuld på 2 miljoner kronor.
```

**Varför det fungerar:**
- Specificitet ger rikare output
- Konflikter skapar dramatisk potential
- Numeriska detaljer gör karaktären trovärdig

### 2. **Iterera, inte kopiera**
AI-output är ett utkast, inte ett färdigt verk.

**Process:**
1. Generera med prompt → 2. Läs och identifiera starka delar → 3. Använd dessa som input för nästa prompt → 4. Upprepa

**Exempel:**
```
Prompt 1: "Skapa en fantasy-värld med två stridande kulturer"
→ Få output med kulturer A och B

Prompt 2: "Utveckla kultur A från föregående värld. Fokusera på deras matriarkala samhällsstruktur och tron på tid som en cyklisk kraft."
→ Få djupare detaljer

Prompt 3: "Skapa en karaktär från kultur A som utmanar traditionerna genom att [specifik handling]"
→ Skapa karaktärer som interagerar med världen
```

### 3. **Kombinera prompts**
De 5 grundprompts är byggstenar.

**Kraftfulla kombinationer:**

| Kombination | Resultat |
|------------|----------|
| Karaktärsutvecklaren + Dialogtränaren | Karaktärer med unika röster och autentisk dialog |
| Världsbyggaren + Berättelsegeneratorn | Berättelser djupt rotade i sin värld |
| Alla 5 i sekvens | Kompletta berättelsekoncept från grunden |

**Exempel på arbetsflöde:**
1. Använd **Världsbyggaren** för att skapa en unik setting
2. Använd **Karaktärsutvecklaren** för att skapa invånare
3. Använd **Berättelsegeneratorn** för att sätta igång handling
4. Använd **Dialogtränaren** för att förfina samtal
5. Använd **Skrivkampslösaren** när du fastnar

### 4. **Anpassa för din röst**
AI kan imitera stilar, men din unika röst kommer från redigering.

**Redigeringschecklista:**
- [ ] **Ton**: Matchar tonen din vanliga röst?
- [ ] **Tempo**: Är meningslängden och pacing som du skriver?
- [ ] **Ordval**: Använder du dina signaturfraser eller ord?
- [ ] **Perspektiv**: Håller den ditt favoritperspektiv konsekvent?

**Övning**: Generera samma prompt 3 gånger, ta det bästa från varje version, och kombinera till något som låter som dig.

### 5. **Bryt reglerna**
När du behärskar grunderna, experimentera.

**Avancerade tekniker:**

**A) Prompt-chaining:**
```
"Skapa en lista med 10 unika magiska system. Välj sedan system #3 och utveckla det till en komplett värld."
```

**B) Constraints som creativity-boosters:**
```
"Skapa en berättelse där varje mening måste innehålla en färg, men ingen färg får upprepas."
```

**C) Meta-prompts:**
```
"Analysera min skrivstil från följande text och skapa en prompt som genererar text i samma stil: [DIN TEXT]"
```

## Vanliga frågor (FAQ)

### **Vilken AI fungerar bäst med dessa prompts?**
Alla moderna AI-verktyg fungerar, men:
- **ChatGPT (4+)** - Bäst för kreativt skrivande och längre output
- **Claude** - Stark på struktur och karaktärsdjup
- **Copilot** - Bra för korta, fokuserade outputs
- **Gemini** - God allround, speciellt med Google-integration

### **Hur många ord bör jag förvänta mig?**
- Grundprompts: 300-800 ord
- Karaktärsprofiler: 200-500 ord
- Världsbeskrivningar: 400-1000+ ord
- Dialogscener: 150-400 ord

**Tips**: Be om specifika ordantal om det är viktigt: "Skriv en 500-ordig berättelse..."

### **Vad gör jag om outputen är för generisk?**
1. **Lägg till begränsningar**: "Undvik klischeer som..."
2. **Specifika exempel**: "Som i [BOK/FILM] men med..."
3. **Tvärtom-prompt**: "Skapa motsatsen till en typisk..."
4. **Kombinera oväntade element**: "En noir-detektiv i en solarpunk-värld"

### **Hur hanterar jag AI:s bias eller repetitioner?**
- **Rotera adjektiv**: Be om "3 alternativa beskrivningar"
- **Byt perspektiv**: "Beskriv samma scen från antagonistens perspektiv"
- **Använd seed-fraser**: "Börja med 'Regnet hade inte upphört på tre veckor...'"
- **Manuell redigering**: Alltid nödvändig för att polera

## Felsökning

### **Problem: AI ignorerar vissa instruktioner**
**Lösning**: 
1. Placera viktiga krav i början av prompten
2. Använd numrerade listor istället för stycken
3. Bekräfta: "Förstår du att du måste inkludera X, Y och Z?"

### **Problem: Output är för kort**
**Lösning**:
1. Specificera ordantal
2. Be om expansion: "Utveckla punkt 3 med 3 underpunkter"
3. Använd "Continued": "Fortsätt från där du slutade och lägg till..."

### **Problem: Karaktärerna låter likadana**
**Lösning**:
1. Ange distinkta bakgrunder för varje karaktär
2. Be om dialogprover innan hela scenen
3. Använd: "Ge varje karaktär en unik talrytm och ordval"

## Avancerade Tips

### **Temperatur-inställningar**
- **Högre temperatur (0.8-1.2)**: Mer kreativa, oväntade outputs
- **Lägre temperatur (0.2-0.5)**: Mer fokuserade, konsekventa outputs
- **Experimentera**: Använd högre temp för idégenerering, lägre för revision

### **Systemprompts för bättre resultat**
Före din prompt, lägg till:
```
Du är en erfaren författare med 20 års erfarenhet av [GENRE]. Du är känd för dina [EGENSKAPER] och din förmåga att [FÄRDIGHETER]. Skriv i en [STIL] stil.
```

### **Spara och organisera dina bästa prompts**
Skapa ett bibliotek med:
1. **Vinnare**: Prompts som genererat ditt bästa material
2. **Kombinationer**: Effektiva prompt-sequences
3. **Genre-specifika**: Anpassade versioner för olika genrer
4. **Redigeringshjälp**: Prompts för revision och polish

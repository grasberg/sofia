# Roadmap: Modernisering av regeringen.se

## Prioriterad backlog (baserat på Gudruns analys)

### 1. Förbättra sökfunktionaliteten
- Implementera smart sök (autoföreslå, filter)
- Lägg till "populära sökningar" på startsidan

### 2. WCAG 2.1-efterlevnad
- Granskning av kontrastförhållanden
- Uppdatering av alt-texter för bilder

### 3. Förenkla startsidan
- Minska navigeringsalternativ
- Implementera kategorisering efter användarbehov

### 4. Språkarbete
- Tillämpa "plain language"-principer
- Strukturera om innehåll med tydlig rubrikhierarki

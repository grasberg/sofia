# Vercel Analytics - Funktions- och Prisanalys

## Sammanställning av webbforskning (2026-03-19)

### Funktioner

**Web Analytics (grundläggande)**
- Spårning av händelser (pageviews, custom events)
- Realtidsanalys med live-uppdateringar
- Geo-analys för besökares plats
- Enkelt gränssnitt i Vercel Dashboard
- Integration med Vercel-projekt

**Web Analytics Plus (tillägg)**
- UTM-parameter spårning
- Ökad datalagring (längre retention)
- Ytterligare avancerade funktioner

**Övriga funktioner**
- Monitoring och observability (för serverless functions)
- Query monitoring (för databaser)
- Performance insights

### Prismodell

**Vercel Plans**
1. **Hobby** – Gratis för personligt, icke-kommersiellt bruk
2. **Pro** – $20 per användare/månad (inkluderar baskvot events)
3. **Enterprise** – Anpassad prissättning med avancerade funktioner

**Web Analytics Events Pricing**
- Pris baseras på antal samlade events över alla projekt
- **Ny prissättning** (efter reduktion med upp till 80%):
  - Tidigare: $2 per 10,000 events
  - Nu: Lägre pris (exakt siffra varierar baserat på volym)
- Inkluderade events i Pro-planen

**Web Analytics Plus Add-on**
- Frivilligt tillägg för ökad retention och UTM-parametrar
- Pris separat från bas-events

### Relevans för Digital Produktpipeline

**Fördelar**
1. **Enkel integration** med Vercel-hostade projekt
2. **Realtidsdata** för snabb insikt
3. **Kostnadseffektiv** efter prisreduktionen
4. **Smidigt** för mindre team och startups

**Begränsningar**
1. **Vercel-låsning** – endast för Vercel-projekt
2. **Event-baserad prissättning** kan bli dyrt vid hög trafik
3. **Begränsad anpassning** jämfört med dedikerade analytics-verktyg

### Rekommendationer

**För vår Niche Selection Toolkit:**
- **Hobby/Pro-plan** kan vara tillräckligt under utvecklingsfasen
- **Web Analytics** ger tillräcklig insikt för MVP
- **Överväg Plus-tillägget** om UTM-spårning behövs

**Alternativ att överväga:**
- Google Analytics (gratis, mer omfattande)
- Plausible (privacy-focused, enkel)
- Fathom (liknande Vercel Analytics men plattformsoberoende)

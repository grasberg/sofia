Google Tools (gog CLI)

Gmail, Google Drive, Google Calendar integration

Setup Instructions

  1. Clone the repo: git clone https://github.com/steipete/gogcli.git then cd gogcli && go build -o gog . && mv gog /usr/local/bin/
  2. Run gog auth login to authenticate with Google
  3. Enable the toggle above and save. Restart Sofia after saving.
  4. Sofia will now have access to Gmail, Drive and Calendar tools.

Brave Search

Web search via Brave Search API

Setup Instructions

  1. Go to brave.com/search/api and create an account
  2. Get your API key from the API Dashboard
  3. Paste the API key above, enable the toggle, and save. Restart Sofia.
  4. Sofia can now search the web using brave_search

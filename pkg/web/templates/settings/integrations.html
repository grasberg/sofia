Integrations

External service tools that require API keys or setup to activate.

Google Tools (gog CLI)

Gmail, Google Drive, Google Calendar integration

Setup Instructions

  1. Clone the repo: git clone https://github.com/steipete/gogcli.git then cd gogcli && go build -o gog . && mv gog /usr/local/bin/
  2. Run gog auth login to authenticate with Google
  3. Enable the toggle above and save. Restart Sofia after saving.
  4. Sofia will now have access to Gmail, Drive and Calendar tools.

Brave Search

Web search via Brave Search API

Setup Instructions

  1. Go to brave.com/search/api and create an account
  2. Get your API key from the API Dashboard
  3. Paste the API key above, enable the toggle, and save. Restart Sofia.
  4. Sofia can now search the web using brave_search

Domain Names (Porkbun)

Register domains, manage DNS, check availability via Porkbun API

Setup Instructions

  1. Go to porkbun.com/account/api and enable API access
  2. Create an API key pair — you will get both an API Key and a Secret API Key
  3. Paste both keys above, enable the toggle, and save. Restart Sofia.
  4. Sofia can now check domain availability, register domains, manage DNS records, and more using domain_name

GitHub Integration

Manage repos, issues, PRs, workflows, and releases via GitHub CLI

Setup Instructions

  1. Install GitHub CLI: brew install gh (macOS) or cli.github.com
  2. Authenticate: gh auth login
  3. Enable the toggle above and save. Restart Sofia after saving.
  4. Sofia can now manage PRs, issues, repos, workflows, and more using github_cli

Note: auth, config, and secret commands are blocked for security.

cPanel Hosting

Deploy websites, manage domains, databases, and SSL via cPanel UAPI

Setup Instructions

  1. Log in to your cPanel account at https://yourdomain.com:2083
  2. Go to Security → Manage API Tokens and create a new token
  3. Enter the hostname, username, and API token above. Port is usually 2083
  4. Enable the toggle and save. Restart Sofia.
  5. Sofia can now upload files, manage domains, databases, and SSL using cpanel

Capabilities: file upload/list/delete, addon/sub domains, MySQL databases & users, SSL certificates.
